# Fruit-Classification-Using-K-Nearest-Neighbors-KNN-
A practice on fruit classification using K-Nearest Neighbors (KNN)
this project applies the K=Nearest Neighbors (KNN) algorithm to classify fruits based on a set of physical and colore-related  features extracted from a dataset. the dataset contains four fruit classes: Orang, Tangerin, Apple, Lemon.
The objective is to predict the fruit type using four specific features: 
Mass (column 4)
Width (colum 5)
Height (column 7)
Colore Score (column 7)

The workflow includes:
1) Loading and preprocessing the dataset
2) Selection the four target features
3) Training KNN classifiers using odd values of K from 1 to 20
4) Evaluating the effect of feature normalization on model performance
5) Comparing accuracy befor and after normalization (It demonstrate how scalling impacts KNN behavior)
